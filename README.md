# Impact of Stress on Students' Academic Performance

## Project Overview
This project is part of the Cisco Data Analytics Essentials course. It analyzes the relationship between students' stress levels and their academic performance using a real-world dataset of 300 student responses.

## Research Question
**How Does Stress Affect Students' Academic Performance?**

## Objectives
- Investigate the relationship between stress and academic performance.
- Identify the factors contributing to students' stress.
- Explore the impact of sleep, academic workload, and financial stress.
- Provide data-driven recommendations for improving student well-being.

## Dataset
- **Dataset:** Student Mental Health Dataset
- **Records:** 300
- **Source:** Google Dataset Search
